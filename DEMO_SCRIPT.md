# Demo 录制脚本 · AI 小说转剧本

> 时长约 2 分钟 | 画面录制 + 简短旁白

---

## 准备

- 打开 `http://localhost:3002`
- 确认 DeepSeek API Key 已配好
- OBS / 系统录屏打开，全屏录制浏览器窗口

---

## 步骤

| # | 操作 | 旁白（可选） | 时长 |
|---|------|-------------|------|
| 1 | **展示空界面** | "AI 小说转剧本，输入小说，一键生成结构化剧本。" | 5s |
| 2 | **粘贴小说文本**（见下方测试文本）到左栏 | "支持粘贴、上传文件、拖拽三种输入方式。" | 5s |
| 3 | **指一下字数统计** | "实时字数统计，超过三千字自动拦截。" | 3s |
| 4 | **点击 Temperature 预设按钮**（稳定/平衡/创意） | "三种创意模式：稳定、平衡、增强。" | 5s |
| 5 | **点击"生成剧本"** | "点击生成，AI 流式输出剧本。" | 2s |
| 6 | **等待打字机效果**展示流式生成 | "打字机效果，逐字实时渲染。" | 15s |
| 7 | **向上滚一下内容** → 弹出"跟随最新内容"按钮 → 点击 | "手动滚动不抢焦点，点击按钮恢复自动跟随。" | 5s |
| 8 | **点击"停止生成"** | "随时可停止，已生成内容全部保留。" | 3s |
| 9 | **在右栏手动编辑一段文字** | "生成结束后可直接编辑剧本。" | 5s |
| 10 | **点击"导出" → "Markdown"** | "支持导出 Markdown，保留完整格式。" | 3s |
| 11 | **点击"导出" → "纯文本"** | "也支持纯文本，自动去除标记。" | 3s |
| 12 | **展示 StatusBar 状态变化** | "状态栏实时反馈：就绪、生成中、完成。" | 3s |

## 测试用小说文本（约 500 字）

把下面这段复制粘贴到左栏：

```
咖啡馆里的灯光昏黄，窗外的雨下了一整天。

林晚坐在靠窗的位置，手里捧着一杯已经凉透的美式咖啡。她的目光始终盯着门口，等待着什么人的到来。

门铃响了。

一个穿着黑色风衣的男人推门进来，他的头发被雨水打湿，贴在额头上。他在门口站了几秒钟，扫视了一圈，然后径直走向林晚。

"你终于来了。"林晚的声音很轻，但很稳。

陈默在她对面坐下，没有点咖啡，也没有脱下外套。"我等了三年。"

"我知道。"

"三年里，我每天都在想这一刻。"

林晚从包里拿出一封信，放在桌上，缓缓推到他面前。"这是她留给你的。"

陈默没有看那封信。他的目光一直停留在林晚的脸上。"我不是为了信来的。"

窗外一道闪电划过，随即是低沉的雷声。咖啡馆里的其他客人纷纷朝这边看了一眼，又各自转回去。

林晚的手指微微颤抖。她低下头，长发垂下来遮住了侧脸。

"那你是为了什么？"

陈默没有回答。他从怀里掏出一把钥匙，放在桌上，紧挨着那封信。
```

## 导出结果示例

**MD 导出** 会保留：
```markdown
### 【内景 咖啡馆·日】
**林晚**：你终于来了。
> 林晚的声音很轻，但很稳。
```

**TXT 导出** 会转为：
```text
【内景 咖啡馆·日】
林晚：你终于来了。
（林晚的声音很轻，但很稳。）
```
